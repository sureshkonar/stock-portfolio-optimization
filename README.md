## Stock Portfolio Optimization Tool

A Python-based application that optimizes a stock portfolio using Modern Portfolio Theory.

### Features
- Fetches real-time stock data
- Calculates portfolio risk & return
- Maximizes Sharpe Ratio
- Visualizes optimized asset allocation

### Tech Stack
- Python
- NumPy, Pandas
- SciPy
- yFinance
- Matplotlib

### How to Run
```bash
pip install -r requirements.txt
python main.py
